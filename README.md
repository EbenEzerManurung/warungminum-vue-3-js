
#warungminum-vue-3-js


Using Vue 3 JS, Bootstrap & DB Json server


## Instalasi
#### Via Git
```bash
git clone https://github.com/EbenEzerManurung/warungminum-vue-3-js.git
```

## Cara install
#### Via Git
```bash
npm install
```
#### Run Via Git
```bash
json-server --watch db.json
npm run dev
```
